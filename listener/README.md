# 🎙️ 语音监听模块（listener）

## 模块简介
该模块用于监听用户语音输入，并进行本地语音识别。支持中英文混合语音识别，使用 whisper 或 faster-whisper 实现，适配低性能本地设备。该模块作为整个贾维斯系统的输入入口。

## 核心功能
- 实时监听麦克风输入
- 离线语音识别（无需联网）
- 支持唤醒词触发（可选）
- 多语言自动判断（可选）

## 依赖建议
- `faster-whisper`（推荐）
- `pyaudio` 或 `sounddevice`（用于音频输入）
- `langdetect`（可选）

## 使用方法
在主程序中导入本模块，或独立运行：
```bash
python listener.py
```
